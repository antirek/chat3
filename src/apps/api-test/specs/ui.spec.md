## Модальные окна

### Общее

1. модальные окна должны иметь заголовок

2. в заголовке модального окна - крестик для закрытия модального окна

3. также модальное окно должно закрываться при нажатии Esc

## Колонка

Понятие для построение интерфейса

содержит Заголовок, Фильтр, Пагинацию, Таблицу

### Заголовок

содержит название колонки

может также содержать кнопки Добавить, URL

### Фильтр

форма выбора варианта для фильтрации ниже расположенной таблицы

Содержит а. выпадающее меню с заранее подготовленными вариантами фильтра, б. само поле для фильтра, кнопку Применить

### Пагинация

элемент пагинации таблицы, должен учитывать применения фильтра. Элемент Пагинация должен быть расположен сразу после Фильтра.

### Таблица

отображает результат запроса к API, у части колонок таблицы могут быть элементы сортировки

### Дополнительные элементы

#### Кнопка URL 

вызывает отображение модального окна, в котором отображается URL текущего запроса к API с учетом фильтра, сортировки и пагинации. Модальное окно соответствует общим правилам модальных окон. Есть кнопка Скопировать, при нажатии копирует URL в буфер памяти.
